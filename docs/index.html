Get started

Set up in three steps

About a minute. The Terminal step is needed only once.

1

Download & install

Open the downloaded insomniac.dmg, then drag Insomniac onto the Applications folder. You can eject the disk image afterward.

2

Allow it to open (one time)

This is a free test build (not yet through Apple's paid notarization), so macOS blocks it at first. Open Terminal (⌘ Space → “Terminal” → Return), paste this, and press Return:

xattr -dr com.apple.quarantine /Applications/insomniac.app

Then open Insomniac from Applications — it launches normally from then on.

3

Find it & use it

Insomniac has no Dock icon — it lives in the menu bar at the top-right of your screen. Look for the moon (a welcome guide points right at it on first launch). Click it, flip Stay Awake on, and enter your Mac password when asked.
